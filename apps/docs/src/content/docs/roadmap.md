---
title: Roadmap
description: Planned direction for Ignis.
---

Larger pieces of work planned for Ignis. The order is approximate and no timeline is given.

- **Mobile improvements and PWA support.** Better behavior on small screens, and an installable Progressive Web App for phones and secondary devices.
- **Obsidian version upgrade.** Moving the pinned Obsidian version to 1.13.1 after its public release.
- **ignis-local.** A first working version of the variant that runs the Ignis server inside desktop Obsidian, serving a vault without Docker.
- **A security model for server plugins.** Isolation for the server-side binaries and external programs that server plugins run.
- **Authentication and access control.** Built-in authentication and per-user access control.

Plugin compatibility fixes, small features, and bug fixes are released continuously alongside this larger work.
